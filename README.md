# Oghie Store API

Oghie Store is a Django REST Framework backend for an ecommerce store. It includes JWT authentication, product catalog management, carts, checkout, orders, payments, CMS sections, and admin analytics.

## Local API Base

```text
http://127.0.0.1:8000
```

## Documentation

- [API documentation](docs/API.md)

## Development

Run the Django development server from the `oghie` directory:

```bash
python manage.py runserver
```

The API root at `/` returns a compact list of the main endpoints.
